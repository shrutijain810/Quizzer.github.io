Instructions for Quiz

  • Each Quiz contain 10 questions.
  • Every question contains 4 options.
  • Out of the given 4 options 1 is correct.
  • Each question is worth 1 point
  • For Each correct answer 1 point is awarded.
  • Wrong answer or Unattempted Question do not affect final score.
  • Scored points out of the total are displayed at the end of the Quiz.
  • One can re-attempt the Quiz by clicking on Try Again button.

About Us

Quizzer is a online platform to test your skills.It has MCQ based questions related to many topics. The Website has 10 Questions of each topic related to each topic, each question has 4 options out of which 1 is correct. It also displays score of the user at the last, awarding 1 mark for each correct answer and no negative marking for wrong answers. The website was developed using HTMl ,CSS and JavaScript. The content of the website is taken from trusted sources.
